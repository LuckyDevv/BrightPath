## 🕯️ О проекте

«Светлый Путь» — это веб-приложение для ритуального агентства, которое помогает людям в трудной жизненной ситуации быстро и достойно организовать похороны.  

Проект включает в себя:

- Публичную часть (главная страница, автопарк, каталог товаров, агенты, калькулятор)
- Закрытую CRM-панель для управления заказами, товарами, услугами и сотрудниками
- Систему авторизации с ролями и двухфакторной аутентификацией (2FA)
- Полностью адаптивный интерфейс (мобильные устройства, планшеты, ПК)

Проект разработан как выпускная квалификационная работа и демонстрирует навыки проектирования, вёрстки, программирования на PHP и JavaScript, работы с базами данных и современными веб-технологиями.

## ✨ Возможности

### 👥 Публичная часть

- **Главная страница** с информацией о компании, преимуществах и тарифах.
- **Автопарк** — список автомобилей с фильтрацией, сортировкой и детальными карточками.
- **Ритуальные товары** — каталог гробов, венков, памятников и т.д.
- **Агенты** — страница «Наша команда» с фильтрацией и биографиями.
- **Калькулятор расходов** — построитель сметы с выбором позиций из каталогов и готовыми пакетами (пресетами).
- **Контакты** — карта, телефон, email, соцсети.
- **Правовая информация** — страницы с политиками и пользовательским соглашением.
- **Форма обратной связи** с уведомлениями на email.

### 🔐 Административная панель (CRM)

- **Дашборд** — общая статистика (заказы, выручка, популярные авто/товары).
- **Управление заказами** — просмотр, фильтрация, изменение статусов, история.
- **Управление товарами/авто/агентами** — полноценный CRUD с загрузкой изображений.
- **Управление клиентами** — база заказчиков с историей заказов.
- **Управление пресетами** — создание и редактирование наборов (пакетов услуг) через удобные мультиселекты.
- **Управление администраторами** — создание учётных записей, роли (админ/менеджер/оператор), блокировка.
- **Настройки** — контакты агентства, смена пароля, сброс 2FA.
- **Заявки на консультацию** — обработка заявок с сайта.

### 🔒 Безопасность

- Хеширование паролей (`password_hash` + соль)
- Двухфакторная аутентификация (TOTP, Google Authenticator)
- Защита от CSRF (токены в формах)
- Лимит попыток входа с временной блокировкой
- Привязка сессии к fingerprint (User-Agent + IP + данные браузера)
- Подготовленные запросы (PDO) — защита от SQL-инъекций
- HTTPS (при развёртывании)

## 🚀 Установка и запуск

### Требования к окружению

- PHP >= 8.5 (с расширениями: pdo_mysql, gd, mbstring, xml, curl, zip, intl)
- MySQL >= 8.0
- Apache 2.4 / Nginx
- Composer
- Git

### Инструкция по установке

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/ваш-username/svetly-put.git
   cd svetly-put
2. **Установить зависимости Composer**
   ```bash
   composer install --no-dev --optimize-autoloader
3. **Настроить базу данных**
   - Создайте базу данных в MySQL
   - Импортируйте схему из файлов database/schema
   - Настройте подключение к базе данных в server/lib/Config.php
4. **Настроить веб-сервер**
   - Направьте корень документа (DocumentRoot) в папку проекта.
   - Убедитесь, что модуль mod_rewrite включён (для .htaccess).
5. **Настройте права на папки**
   ```bash
   chmod -R 755 server/cache server/logs
6. **Добавить первого администратора**
   - Через SQL-запрос с хешем пароля (password_hash)
7. **Открыть сайт в браузере**
